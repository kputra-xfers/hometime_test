# Hometime Test

#### Ruby: 3.0.0
#### Rails: 7.0.1
#### Database: Sqlite3

## Running server
1. Clone the repository
2. In terminal, move the directory to the cloned repo
3. Run `$ rails s`

## Running unit test
1. Clone the repository
2. In terminal, move the directory to the cloned repo
3. Run `$ rspec`
